#LinkIt jQuery Plugin#

**LinkIt** is a simple jQuery plugin yo attack links to elements

##Versions##
0.2.0

##Usage##
	$('span').linkIt({
      whatever: 'anything',
      href: 'http://test.com',
      text: 'Click Me',
      target: '_blank'
   });

##Vendors##
jQuery - [http://jquery.com](http://jquery.com)
